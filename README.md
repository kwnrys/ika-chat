# イカチャット 🦑

スプラトゥーン3の遠隔意思共有用チャット。スマホブラウザで動く、その場限りの作戦ボード。

**公開版URL: https://kwnrys.github.io/ika-chat/**（URLを開くだけで誰でも使えます）

## 特徴

- **ワンタップでスタンプ即送信** — 前線上げて／人数有利／2キル！など21種
- **読み上げ機能（🔊）** — プレイ中に画面を見なくても、味方のスタンプを音声で聞ける（受信時バイブも）
- **ログは一切残らない** — メッセージは保存されず中継されるだけ。画面上も8秒で自動消滅
- 4桁のルーム番号で合流（参加者にはスプラ風インクカラーが割り当てられる）
- 自由テキスト入力も可能

## 使い方

1. https://kwnrys.github.io/ika-chat/ をスマホで開く（LINEなどでURLを共有）
2. 誰かが「ルームを作る」→ 表示された4桁番号を仲間に伝える
3. 仲間は番号を入れて「参加」

## 仕組み

- GitHub Pages（静的ホスティング）＋ 公開MQTTブローカー（broker.emqx.io / 予備: test.mosquitto.org）の`retain`なしpub/subで中継。自前サーバー不要
- ⚠️ 公開ブローカーを使うため、ルーム番号（トピック名）を知っていれば理論上は誰でも購読できます。ゲームの作戦連絡以外（個人情報など）には使わないでください
- メッセージはブローカーにも保存されません（retainなし・QoS0）

## 開発メモ

- 本体は `index.html` 1ファイル＋同梱の `mqtt.min.js`。スタンプの追加・変更は `index.html` 冒頭の `STAMPS` 配列を編集
- iPhoneの読み上げは、一度画面をタップした後から有効になります（ブラウザの自動再生制限のため）
